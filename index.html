Architecture

End-to-End Pipeline

Five layers from raw API to interactive dashboard — each stage purpose-built for reliability, reusability, and scale.

01
🌐
Ingest
Python · Requests · Pandas
Fetches World Bank API with batching, pagination, and retry logic. Loads ~9K rows to BigQuery.
→
02
🗄️
Store
Google BigQuery
Partitioned by year, clustered by country and indicator. Raw layer preserved for full reprocessing.
→
03
⚙️
Transform
dbt Cloud
Staging → Dimension → Fact model. Deduplication, type casting, YoY growth calculations.
→
04
🔄
Orchestrate
Airflow · Docker
Weekly DAG runs in Docker containers via WSL2. Ingest → staging → marts in sequence.
→
05
📊
Visualise
Power BI Service
Two-page interactive dashboard. Continental overview and country deepdive with live slicers.
Technology

Tech Stack

Industry-standard tools across every layer of the modern data stack.

Ingestion
Python
Requests, Pandas, PyArrow. Batch API calls with exponential backoff retry logic.
Storage
BigQuery
Range partitioned on year. Clustered on country_iso and indicator_name for query efficiency.
Transformation
dbt Cloud
3-layer model: raw → staging (views) → marts (tables). Sources registered in YAML.
Orchestration
Airflow 2.8
Dockerised via Docker Compose on WSL2. CeleryExecutor with Redis and Postgres backends.
Containerisation
Docker
7-container stack: webserver, scheduler, worker, triggerer, postgres, redis, init.
Visualisation
Power BI
Connected directly to BigQuery via service account. DAX measures for dynamic KPIs.
Version Control
Git / GitHub
Full project tracked with .gitignore, .env.example, and structured folder organisation.

Data Model

dbt Layer Architecture

Three-layer medallion architecture — raw preservation, clean staging, and analytical marts.

Source · Raw
world_bank_indicators
BigQuery Table
Raw ingested data. One row per country × indicator × year. ~9K rows. Append-only.
Staging · Views
stg_world_bank_indicators
View
Deduplicates on ingestion_id. Casts all types explicitly. Filters null country codes.
Marts · Tables
dim_countries
Table
One row per country. ISO code, name, and derived region classification.
fct_economic_indicators
Table
Full fact table with gdp_usd, gdp_yoy_growth_pct (LAG window function), and all indicator values.
